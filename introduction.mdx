# Reward Protocol

**Author, reproduce, and evaluate reward functions seamlessly on Fireworks, TRL, and your own infrastructure.**

Reward Protocol simplifies the creation and deployment of reward functions for evaluating AI model outputs. Whether you're working on math problems, code generation, tool calling, or custom evaluation metrics, Reward Protocol provides the tools you need.

## Integrations

Reward Protocol integrates seamlessly with popular ML platforms and frameworks:

- **Fireworks AI**: Deploy and scale your reward functions
- **TRL (Transformer Reinforcement Learning)**: Use in RLHF training pipelines
- **Braintrust**: Connect with evaluation and monitoring workflows
- **OpenEvals**: Leverage existing evaluation metrics
- **DeepEval**: Integrate LLM-as-a-judge evaluations
